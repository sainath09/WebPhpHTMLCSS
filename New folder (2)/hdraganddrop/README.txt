Features:
---------
a) Slides Open the Image Menus automatically Onmouseover.
b) Simple image rollover menu script with horizontal navigation.
c) You can set the menu to appear onmouseover or rollover the image by the
user.
d) Easy to use javascript horizontal scrolling image menu script.

Installation:
--------------
Please take 5 minutes time and read installation instructions carefully and
completely! This will ensure a proper and easy installation. 

Extracting files:
a) Unzip the hdraganddrop.zip file to extract the files.
b) You will get hdraganddrop/js/dragable-content.js,hdraganddrop/js/jquery.js, hdraganddrop/dragable-content.php, hdraganddrop/README.txt and images.
Embedding the code:
a) Place the following div tag in the page where you want to use this effect.
  <div style="color: #006622;"><h4>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Drag and Drop the images anywhere in this page</h4>
  <p align=center><img src="./images/image1.jpg" style="padding-left: 10px; float:left" class="dragableElement"></p>
  <p align=center><img src="./images/image2.jpg" style="padding-left: 10px; float:left" class="dragableElement"></p>
  <p align=center><img src="./images/image3.jpg" style="padding-left: 10px; float:left" class="dragableElement"></p>
  <br><br><br><br><br><br></div>

Change the image source to your required images.

b) Include dragable-content.js file in the page where to apply this effect. Place the below <script> in your <head> portion,
  <script type="text/javascript" src="js/dragable-content.js"></script>

c) Run "dragable-content.php" file to view the working of this script.
