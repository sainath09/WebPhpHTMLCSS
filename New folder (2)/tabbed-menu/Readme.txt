Features:
---------
a) The contents are hidden inside different divs.
b) Clicking on any tab will show the respective content..
c) This script is a combination of JQuery and customizable CSS tricks.
d) Cool looking tabbed horizontal navigation menu for the websites.
e) Good space saver for wordpress, article and blog relates sites.

Downloads:
-----------
Please visit our site http://www.hscripts.com/scripts/jquery/tabbed-menu.php and do the download

Please take 5 minutes time and read installation instructions carefully and
completely!. 

Extracting files:
a) Unzip the tabbed-menu.zip file to extract the files.
b) You will get  tabbed-menu/index.php,tabbed-menu/jquery.js,tabbed-menu/text/textfile.txt,tabbed-menu/README.txt and images.

Embedding the code:
a) Now Run the tabbed-menu/index.php in your browser.
b) JQuery Tabbed Menu script is now ready to work fast and efficient.
c) If you want to change the content of the tabs, change it in the text/textfile.txt
d) If you want to change the image, then give your image path in the text/textfile.txt
after the "-", which is provided at the end of the description.
